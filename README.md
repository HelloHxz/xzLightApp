# xzLightApp - React

> React LightApp
该项目处于实验阶段，旨在将常用组件通用化<br/>
现在已经正在做的是将大部分App进行高仿来说明该框架的能力<br/>
现已经正在高仿的应用如下</br>
* QQ  
* 微信
* 美团 
* 京东 
* 苹果日历
* 支付宝
* 微博

其中有些只是模仿了其中的一小部分 后续会继续完善<br/>
所有高仿都是基于现有通用组件进行扩展搭建



## Build Setup

# 启动服务
